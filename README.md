# MSema :pencil:

This is a social media 📱 project inspired by twitter 🐦 and discord 

## Getting Started 💻

First, run :runner: the frontend server in base directory:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
Secondly. run 🏃 the backend server in backend directory:
```bash
python manage.py runserver
```
